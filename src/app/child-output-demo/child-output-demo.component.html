Demonstration of Passing Data from Child to Parent Container

  • Instruction
    1. Create a child component and render it in the main component as an HTML tag to see if that works
    2. Create a function in `parent.ts` and call this function in `parent.html` inside the selector with (functionEvent)="callFunction($event)"
    3. Now inside `child.ts`, use the at `Output` decorator and pass the name of the event function to it. Instantiate the EventEmitter like at Output() updateValueEvent = new EventEmitter<string>();
    4. Come to `child.html`, create an input box with an ID, and call the function updateValueEvent.emit(id.value)
    5. To display this value in the main component, go to `parent.html`
